# Changelog

## [1.0.0] - 2025-09-10

### Added
- XMLサイトマップとRSSフィードの解析機能
- XMLバージョン、フォーマットタイプ、件数の自動判定
- XML内URL到達性チェック機能（50件ずつのチャンク処理）
- 定期実行スケジュール機能（cron対応）
- メール通知機能（チェック結果の自動送信）
- 緊急停止・セッションクリーンアップ機能
- 管理画面UI（URLリスト設定、スケジュール設定）
- セキュリティ対策（管理者権限、nonce検証、サニタイズ処理）

### Fixed
- 管理画面の保存ボタン位置を改善（スケジュール設定の下に配置）
- URLリスト説明文を明確化（XMLサイトマップとRSSフィードの説明追加）

### Improved
- XMLアナライザーでの複数フォーマット対応（Sitemap Index、URL Set、RSS、Atom）
- cURL使用時のタイムアウト・エラーハンドリング強化
- 大量URL処理時のメモリ使用量最適化
- 日本時間対応のスケジュール機能
